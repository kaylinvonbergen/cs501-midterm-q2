# Q2 ViewModel Counter

## Description
This project implements a simple counter screen using Jetpack Compose with a ViewModel.  
The counter value is stored in a ViewModel and updates the UI through recomposition when changed.  
A button is provided to increment the count.

I initially wrote a version of the `CounterScreen` composable without any centering. That version is commented out in the code. I then rewrote the same function (the active version) with proper centering so I could better visualize and test the layout on screen.

The commented version is included below for clarity and comparison.
